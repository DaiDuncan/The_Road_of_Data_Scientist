# 6.动词

## 6.1 动词的意义

表示**人或事物**的**动作、行为**或**状态**

## 6.2 动词的分类

| 1）独立动词     | 单独或加补足语作谓语                                         | laufen-Ich  laufe.  schlafen-Ich  schlafe.                   |                                                              |
| --------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 2）助动词       | 不表示动作或状态；  只与独立动词构成复合时态时起辅助作用  == 时间助动词 | 现在完成时  haben-Ich  habe gelernt.  sein-Ich  bin gelaufen.  将来时  werden-Ich  werde laufen. | 三者也可以作独立动词用  Ich habe  ein Buch.(besitzen)  Ich bin  da.(existieren)  Wir sind  drei.(vorhanden sein)  Die Tage werden länger.(sich verlängern) |
| 3）**情态动词** | 表示某种能力-können  必要性-müssen  可能性-können  愿望-wollen/möchten  意志-wollen  猜测-können  允许-dürfen  建议-sollen  劝告-sollen  爱好-mögen  等**情态**意义  一般与独立动词连用，构成复合谓语 |                                                              | 也可作独立动词  Ich kann  Deutsch.(beherrschen)  Er will das.(wünschen)  Du sollst  nicht.(verpflicht sein)  Wir dürfen es.(gestattet sein)  Ihr müsst so.(notwendig sein)  Sie mögen es.(lieben) |
| 4）无人称动词   | 天气和自然现象；  多用es作主语                               | regnen  schneien  donnern  blitzen  frieren-寒冷  reifen-下霜  tauen-化雪/下露水  stürmen-刮风暴  dämmern-黎明/日暮  grauen-破晓  tagen-天亮  dunkeln-天黑 |                                                              |
| 5）**反身动词** | 行动涉及对象为主语本身，通过反身代词sich来体现               | 真反身动词  必须与反身代词连用（不可用名词等替代）  Wir müssen uns beeilen. | 假反身动词  可用反身代词替代(名词)宾语  Der  Rechtsanwalt verteidigt **den Angeklagten**.别人**替**辩护  Der  Angeklagte will**sich** verteidigen.自我辩护 |

## 6.3 动词的变位

3种人称；单复数；6个时态；

### 6.3.1 动词的三个基本形式

| 不定式   | fragen  | sagen  | gehen    |
| -------- | ------- | ------ | -------- |
| 过去时   | fragte  | sagte  | ging     |
| 过去分词 | gefragt | gesagt | gegangen |

### 6.3.2 动词的时态 die Tempora

#### 6.3.2.1 动词时态的形式

#### 6.3.2.2 动词时态的用法

| 简单形式  现在时     | **说话时(目前)**正在发生的动作、行为或**目前**的**状态**     | Ich frage  den Lehrer.                                       | Ich gehe  zur Arbeit.         |
| -------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ----------------------------- |

#### 6.5.2.4 虚拟式的用法

| 第一虚拟式的用法(der Gebrauch des Konjunktivs I)             |                                                              |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| 1）表达**有可能**或希望实现的**愿望**                        | Es lebe die  Freundschaft zwischen China und Deutschland!  Sie möge kommen! |
| 2）用于**间接引语**或间接问句                                | Er sagt, **dass** du Frau Moll nicht gegrüßt habest.  Sie fragt  ihn, **wie viel** **Geld** er fordere.  Sie fragte  ihn, **ob** er ihr das Auto nicht  verkaufen wolle. |
| 3）用于指示或**间接命令**(主语为man)，常见于说明书及菜谱中。  现代德语中用不定式代替表示 | Man nehme dreimal täglich eine Tablette.  Man lasse den Tee fünf Minuten ziehen!  Bei Gefahr Scheibe **einschlagen** und Knopf **drü****cken**. |

## 6.9 动词的配价

### 6.9.1 动词配价的意义

动词对特定成分的支配和许可能力。

配价成分 == 补足语，与配价载体 == 动词存在依存关系

动词及其补足语是组成句子用来表达的最小单位

### 6.9.2 动词补足语类别
