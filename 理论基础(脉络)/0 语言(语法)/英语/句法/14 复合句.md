# 14 复合句

主句+连词+从句；

连词+从句+主句；

主句是句子的主体，从句从属于主句并充当主句的一个成分，分为名词性、定语和状语从句

## 14.1 定语从句

### 14.1.1 定语从句的定义

在复合句中，修饰某一名词或代词的从句。The **story** **that** you read is *The Rescue*.

放在先行词story的后面。

关系代词或关系副词在定语从句中**充当一个成分**

### 14.1.2 由关系词引导的定语从句

| 关系词            | 先行词     | 担当成分   | 举例                                                         | 备注                                                         |
| ----------------- | ---------- | ---------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 关系代词  who     | 人         | 主语       | This is the  man who helped me.                              | whom及that,  which在**从句中作宾语**时，口语和非正式文体中**常可省略**；  但**介词提前时**，that不能用；  此时，whom,  which不能省略；  **whose** ==  of whom指人  **whose** ==  of which  指物 |
| whom(who-宾格)    | 人         | 宾语       | The girl  (whom) I met looks like Lily.                      |                                                              |
| that              | **人、物** | 主语、宾语 | A plane is  a machine that can fly.                          |                                                              |
| which             | 物         | 主语、宾语 | This is a  book which you want.                              |                                                              |
| whose(who-所有格) | **人、物** | **宾语**   | The room  whose window is red is mine.                       |                                                              |
| 关系副词  when    | 时间       | 状语       | It happened  on the day when I was out.                      | when==  in/on which                                          |
| where             | 地点       |            | I recently  went to the town where(= in which) I was born.   | where==in/at  which                                          |
| why               | 原因       |            | The reason  why(= for which) he didn't come was that he missed his train. | why== for  which                                             |

### 14.1.3 由介词+关系代词引导的定语从句

 

| 介词+关系代词                  |
| ------------------------------ |
| 1 由于**先行词**的要求         |
| 2 由于**从句**某个关键词的搭配 |
| 3 出于**语意表达**的需要       |

 

| 介词位于**关系词**前       | which,  whom作介词的宾语                                     | Mr. Li is the person **from whom** you should learn.         |
| -------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 介词位于**从句有关动词**后 | 关系词如果是作“动词+介词”固定词组的宾语  介词一般不拆开，放在动词后面 | The baby(whom/who/that) the nurse **looks** **after** is healthy. |

### 14.1.4 限制性定语从句与非限制性定语从句的区别

| 限制性   | 限制性定语从句  是先行词在意义上不可缺少的定语；从句与主句**关系密切**，不用逗号分开 | She has  found the necklace **(that)** she  lost two weeks ago. |
| -------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 非限制性 | 从句与主句关系不十分密切，往往用逗号分开，一般不用that引导   | Yesterday I  met Li Ping,**who** looked very tired.          |

### 14.1.5 that与which/who(m)的用法区别

| 可互用的情况  which与that；  who(m)与that；              | 1  限制性定语从句中，先行词是物时，which与that可互换  2 限制性定语从句中，先行词是人时，who(m)与that可互换 |
| -------------------------------------------------------- | ------------------------------------------------------------ |
| 只用which,  who(m)的情况                                 | 1 **非限制性定语从句**中，指物只用which；  指人用who/whom。  She was  awarded a gold medal, which the whole family considered a great honor.  2 “**介词+关系代词**”引导的定语从句中，指物用which；指人用whom。  3 **先行词本身**是that时，关系词用which；  先行词为those,  one, ones时，多用who。 |
| 只用that的情况  (不定代词，不定形容词，带限制意味的词等) | 1 all, everything, anything,nothing(something除外), little, much等**不定代词**作先行词时  You must do  everything that I do.  2  先行词被序数词或形容词最高级以及first, last, only, very等词修饰时  James is  the best singer that I have ever known.  3  先行词既指人又指物时  We talked  about the things and persons that we remembered in school.  4 先行词被the  only, the very修饰时  She is the  only person that I want to see.  5 先行词被all,  any, every, each, few, little, no, some等**修饰**时  I’ve read  all the books(that) you gave me.  6  当句中已有who时，为避免重复**@德语**  Who is the  person that is talking to our headmaster? |



## 14.2 名词性从句

### 14.2.1 名词性从句种类及作用

| 主语从句                                                     | ==  名词，担当主句的主语  一般置于谓语动词前，但多数情况用it作形式主语，避免头重脚轻 | 【**连接词**】   that,  whether, if  (仅用于宾语从句)  as if  (仅用于表语从句)     【连接**代词**】  who, whose,  what, which     【连接**副词**】  how, when,  where, why    #注意  1  除了引导宾语从句的that在口语中省略，其他引导词均不可省略  2  注意引导词在从句中担当的成分以及有无词义，有什么词义等 | **Who did that** is known  to all.            |
| ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ | --------------------------------------------- |
| 表语从句                                                     | ==  名词，担当主句的表语  置于主句系动词后                   |                                                              | It **looks** **as if** it  is going to rain.  |
| 宾语从句    that(口语中常省略), whether, if    连接代词   what,  which, who, whom    连接副词   when,  where, how, why | ==  名词，担当主句的宾语  置于vt.或介词后                    |                                                              | We can **learn****what** we  did not know.    |
| 同位语从句                                                   | 一般置于某些抽象名词后，具体说明该名词内容  news,  truth, ides, fact, advice, thought, promise, belief等 |                                                              | You have no **idea****how  worried she was**! |

 

### 补充|解释

1）宾语从句-语序的变化

| a.  陈述句变为宾从，语序不变                                 | He is an  honest man.  -I said **he was an  honest man**.    |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| b.  一般疑问句，变为陈述句语序, whether,if连接               | Does he  study hard?  -I wonder if(whether) **he studies hard**. |
| c. 特殊疑问句,  变为陈述句语序, 用连接代词/副词  (保留原来的疑问词) | Who is he?  -Do you **know** **who** **he is**?  Where will  he arrive?  -He **asked** me **where** **he would arrive**. |

### 14.2.2 同位语从句与定语从句的**区别**

| 同位语从句                                                   | 定语从句                                                     |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| 阐明前面名词的具体内容  The news  that he won is true.       | 限定前面的名词的**范围**或补充一些情况  The news  (that) he told me is true. |
| 常用that引导，that为连词  在从句中**不担当**成分，但不可省   | 由关系代词that或which引导  在**从句中**担当成分，如主语、宾语  作宾语时，口语中常省去 |
| 只有在内容方面可进一步挖掘/阐明的名词  一般为抽象名词  news, fact,  idea, question, suggestion | **几乎所有名词**都可带定语从句                               |
