# [酷python](http://www.coolpython.net/hibiscus/python_stu.html)

**每人1500**，项目周期初步定为2个月(1-3个月)



## 技术

|            | 技术           |
| ---------- | -------------- |
| web开发    | flask框架      |
| 爬虫开发   |                |
| 数据库     | mysql, mongodb |
| 消息队列   | redis          |
| 自动化部署 | fabric         |



按照企业开发项目的一般流程工作：

1. 学会使用git协同合作
2. 学会使用各种python框架
3. 参与体验系统开发流程，实打实的开发经验
4. 学习了解项目架构
5. python项目部署，维护
