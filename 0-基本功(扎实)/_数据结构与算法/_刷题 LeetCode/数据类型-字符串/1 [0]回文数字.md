思路1：转化为字符串

```python
class Solution(object):
    def isPalindrome(self, x):
        """
        :type x: int
        :rtype: bool
        """
        str_x = str(x)
        if str_x[::-1] == str_x:
            return True
        else:
            return False
```



思路2：[7]Reverse Int

- 68ms
- 时间：34.4%
- 空间：13.82%

```python
class Solution(object):
    def isPalindrome(self, x):
        """
        :type x: int
        :rtype: bool
        """
        if x < 0:
            return False

        origin = x
        res = 0
        while(x != 0):
            remainder = x % 10
            x = x // 10

            res = res * 10 + remainder
        
        if res == origin:
            return True
        else:
            return False
```
